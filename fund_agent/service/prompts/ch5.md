【数据验证规则 - 必须严格遵守】

【口径强调 - 必须严格遵守】
数据表中有两个不同口径，不可混淆：
- **权益投资规模变动（代理指标）**：年报资产配置中权益投资金额同比变动，仅供参考，不用于阈值判定
- **规模变动阈值判定**：必须使用「份额×净值同比」口径，数据表中如有预计算结果则直接引用

【阶段判定 - 逐步核对】
按以下顺序逐步检查，每一步都必须引用数据表中的具体数字：
1. 建仓期：基金成立是否 < 2 年？
2. 转型期：是否有基金经理变更 / 投资目标变更 / 业绩基准变更 / 基金类型变更？
3. 膨胀期：份额×净值同比是否 > 50%？
4. 萎缩期：份额×净值同比是否 < -30%？
5. 以上均不满足 → 稳定期

每一步必须写出：检查了什么数据 → 结果是什么 → 是否触发。
