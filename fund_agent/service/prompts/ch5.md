请基于上述数据和阶段判定规则，写一段「当前阶段与关键变化」分析。

分析逻辑链：数据 → 阈值 → 触发 → 影响 → 跟踪。不可跳过中间环节。

【强制输出结构 — 必须按以下 4 段顺序输出，每段必须有 ## 标题，不得跳过任何段落】

## 当前阶段

从数据表「阶段判定规则」节直接引用判定结果和判定依据中的具体数字。
- 如果判定依据为"默认"或"代理指标"，必须如实转述，不得改写为确定性结论
- 如果数据表无「阶段判定规则」节，输出「阶段判定数据不可用」并终止本章

## 关键变化

从持仓变动、规模变动、费率变动 3 个维度逐项检查。每项必须输出：
- 维度名称、数据表中的具体数值（原样引用）、是否触发阈值（是 / 否 / 数据不可用）
- 触发时说明变化方向与幅度
- 数据不可用时使用固定话术：「[维度名] 数据不可用（[原因]），该维度不纳入关键变化」

三项检查完毕后汇总触发项作为「最关键的变化」。零触发时输出：
「当前年度无阈值触发的关键变化，基金处于 [阶段名] 的正常波动范围。」

## 投资假设影响

- 数据表包含信号评分：对比评分方向是否逆转，引用具体指标和分值
- 数据表不包含信号评分：使用固定话术「信号评分数据未包含在数据表中，投资假设影响无法通过程序化方式判定」
- 禁止凭空推断信号评分方向，禁止写"信号评分可能发生了变化"等猜测性表述

## 跟踪变量

逐条列出最该跟踪的 1-3 个变量，每条包含变量名、为什么盯它、向好 / 向坏分别会改变什么判断。
无法从数据表确定时使用固定话术：「当前数据不足以确定优先跟踪变量，建议下一轮补充 [具体缺失数据] 后再评估。」

【口径区分 — 必须严格遵守】
- 权益投资规模变动（代理指标）：仅描述趋势，不用于阈值判定
- 份额×净值同比：用于阈值判定（>30% / <-30%）
- 两个口径不可混用或互相替代。出现"权益投资规模变动"时必须附带"（代理指标，详见数据表口径说明）"

【数据缺失降级话术 — 遇缺失必须使用原话】
- 份额×净值同比缺失 → 「规模变动阈值无法判定（口径数据缺失），以下分析仅基于权益投资规模变动代理指标」
- 前十大持仓换手率缺失 → 「持仓变动维度数据缺失，该维度不纳入关键变化筛选」
- 费率数据仅单年 → 「费率变动维度仅有一个年度数据可用，同比变动无法计算」
- 信号评分数据缺失 → 「信号评分数据未在数据表中提供，投资假设影响判定跳过」

【禁止项 — 以下任一情况视为违规】
- 输出操作建议（买入 / 卖出 / 持有 / 加仓 / 减仓 / 替换）
- 输出预测性表述（"预计未来""后市将""大概率""目标价"）
- 罗列超过 3 个关键变化
- 将权益投资规模变动的数值直接用于阈值判定
- 数据缺失时自行推算或编造数字
- 使用"约 XX 亿"等模糊数字替代原始精度数字
